# 行为进度条项目

一个非常简单的小工具：给自己设定一个行为（例如「每天背单词」「每天运动 20 分钟」），每做一次就点一下按钮，让进度条带动画往前走，直观看到自己离目标还有多远。

## 技术栈

- 后端：Spring Boot 3（`spring-boot-starter-web`），使用内存保存当前配置和进度
- 前端：纯静态 HTML + CSS + 原生 JS，直接由 Spring Boot 静态资源目录提供

## 主要功能

- 在页面上设置：
  - 行为名称（例如「学习英语 30 分钟」）
  - 每次点击增加的百分比（例如 5%）
- 每完成一次行为，点击「完成一次行为」按钮：
  - 调用后端接口记录一次
  - 进度条宽度带动画增加相应比例
  - 显示当前百分比和已完成次数
- 进度到 100% 代表当前阶段目标完成，可以重新配置新的目标

## 运行方式

1. 确保本机安装了 JDK 17+ 和 Maven。
2. 在项目根目录执行：

```bash
mvn spring-boot:run
```

3. 启动成功后，在浏览器打开：

```text
http://localhost:8080/
```

前后端会一起启动和提供服务，部署只需要打包一个 jar 放到服务器上运行即可。
